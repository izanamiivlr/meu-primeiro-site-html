# 🌙 Login Izanamii

Tela de login fictícia, criada como exercício de estilização com HTML e CSS.

🔗 **Link do site:** https://izanamiivlr.github.io/meu-primeiro-site-html/

## Sobre o projeto

Esse projeto é uma interface de login fictícia, com clima de RPG noturno — uma floresta pixel art iluminada por lanternas e luz de lua, com um card de login flutuante sobre o cenário.

Foi o meu primeiro projeto estudando HTML e CSS, criado para praticar:

- Estrutura de formulários (`<input>`, `<label>`, `<button>`)
- Estilização de campos de usuário e senha
- Efeito de cartão flutuante sobre uma imagem de fundo
- Centralização e organização de elementos com CSS

## Tecnologias usadas

- HTML5
- CSS3

## Como visualizar localmente

1. Clone este repositório:
   ```bash
   git clone https://github.com/izanamiivlr/meu-primeiro-site-html.git
   ```
2. Abra o arquivo `index.html` no navegador.

## Observação

Este é um projeto **fictício**, feito apenas para fins de estudo — não há autenticação real nem coleta de dados.
